    라이브 불러오기 설정 (CORS 프록시)

    아이티동스쿨은 브라우저의 교차출처(CORS) 직접 호출을 허용하지 않습니다. 강좌 목록을 페이지 안에서 실시간으로 보려면 CORS 프록시가 필요합니다.

    가장 안정적인 방법은 본인 Cloudflare Worker를 띄우는 것입니다 (저장소의 cors-proxy-worker.example.js 참고). 그 주소를 아래에 입력하세요. 비워두면 공개 프록시 자동 시도 → 실패 시 공식 페이지 링크로 대체합니다.
